## 🛠️ Développement local

### Prérequis
- Git
- Go 1.21+ (optionnel pour la lecture)
- Node.js (pour docs.page CLI)

### Installation
```bash
# Cloner le projet
git clone https://github.com/username/go-docs.git
cd go-docs

# Option 1: Avec docs.page CLI
npm install -g @docs.page/cli
docs.page serve

# Option 2: Serveur HTTP simple
python -m http.server 8000

# Option 3: Live Server (VS Code)
# Installer l'extension "Live Server" et clic droit > "Open with Live Server"
```

### Accès local
- **docs.page** : http://localhost:3000
- **HTTP simple** : http://localhost:8000
- **Live Server** : http://127.0.0.1:5500

### 🔄 Comment contribuer

1. **🍴 Fork** le projet
2. **🌿 Créez** votre branche (`git checkout -b feature/amelioration`)
3. **✍️ Commitez** vos changements (`git commit -m 'Ajout: nouvelle section sur...'`)
